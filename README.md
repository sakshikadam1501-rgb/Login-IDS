# 🔐 Login Intrusion Detection System

## 📌 Overview
This project is a Machine Learning-based Login Intrusion Detection System that detects suspicious login attempts in real time. It uses behavioral features such as login time, IP address, and failed login attempts to classify users as **Normal** or **Intruder**.

---

## 🚀 Features
- Real-time login detection
- Machine Learning (Random Forest)
- Rule-based intrusion detection
- Web-based UI using Flask
- SQLite database for storing login logs

---

## 🧠 How It Works
1. User enters login details (IP, time, failed attempts)
2. Data is sent to Flask backend
3. Machine Learning model predicts user behavior
4. Rule-based logic validates high-risk attempts
5. Result is displayed (Normal / Intruder)
6. Data is stored in database
